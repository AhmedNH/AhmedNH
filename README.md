## 📌 Featured Projects

### 🔹 ShareU (Java, Spring Boot, Spring Security, JWT, SQLite, React, TypeScript, Vite)
Full-stack peer-to-peer item sharing platform for York University students. Spring Boot 4 REST API backend with stateless JWT authentication via Spring Security, HttpOnly cookies, and university email domain validation. React 19 + TypeScript frontend with TanStack Router, modular CSS, and a custom API cache layer. Layered architecture with controllers, services, and repositories, OpenAPI documentation, and a GitHub Actions CI pipeline with unit and integration tests.

### 🔹 Flux: Scripting Language Interpreter (C)
A fully functional scripting language interpreter built from scratch in C — no parser generators or external libraries. Hand-written lexer, recursive-descent parser, and tree-walking evaluator across ~1,000 lines of modular C. Implements lexical block scoping via a linked-list scope chain, manual memory management with deep-copy value ownership, recursive function support, and an interactive REPL with multi-line input buffering.

### 🔹 Pulse: Data Aggregation Pipeline (Python, FastAPI, SQLite)
Real-time data pipeline that ingests weather and cryptocurrency market data from two public APIs, stores time-series snapshots in SQLite with WAL journaling and composite indexing, and exposes results through a FastAPI REST server and a Rich terminal dashboard. Implements thread-local DB connections, exponential-backoff retry logic, and a configurable multi-job scheduler. Covered with 17 unit tests using in-memory SQLite and fully mocked HTTP.

### 🔹 Virtual Memory MMU Simulator (C)
Software-based MMU that translates logical addresses to physical addresses via bit masking. Implements a TLB with FIFO replacement, demand paging from a binary backing store, and LRU page replacement to resolve page faults under constrained physical memory. Reports page fault rate and TLB hit rate.
